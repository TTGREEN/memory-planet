## 🔑 关键技术发现

### 1. Sign 签名（已确认）
```
sign = MD5(appKey + timestamp + token + data).toLowerCase()
```
- **appKey**：淘宝 12574478（1688 待实测）
- **timestamp**：13 位毫秒
- **token**：`_m_h5_tk` 的第一段
- **data**：JSON 字符串（无空格）

### 2. Cookie 刷新机制（已确认）
```
_m_h5_tk 格式：token_t_random
生命周期：每次请求后从 Set-Cookie 更新
2025-09 后：有效期 → 15 分钟
```

### 3. 新增反爬（2025，待逆向）
- **x5sec**：类似 _m_h5_tk_enc 的加密字段
- **acsign**：额外签名层（算法未知）
- **utdid**：22 位设备指纹（需生成并保持）

### 4. 性能优化（已验证）
- **Node.js crypto**：`crypto.createHash('md5')`
- **对比 execjs**：快 3.74 倍（无 JVM 启动开销）
