## 教训
- PowerShell 中 `&&` 不是语句分隔符，用 `;` 或换行
- Star-soul-core-runner.js 用 `&&` 会导致整个命令解析失败，用 `-and` 逻辑符或分号
- `node -e` 中 heredoc 多行要小心 PowerShell 解析，改用 .cjs 文件更可靠
- MiniMax API 的 x-api-key header 是正确的，不是 Authorization: Bearer
