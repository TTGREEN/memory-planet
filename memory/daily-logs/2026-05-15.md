来源：仕泽提供的 GitHub 链接 https://github.com/D4Vinci/Scrapling

- 创建 LEARNINGS.md（7601字节，完整架构分析）

### 核心发现
1. **Parser 层完全独立**：只需要 lxml，可以直接用 Selector 做 HTML 解析，不需要浏览器
2. **find_similar** 的相似度算法（SequenceMatcher）可以迁移到 recall 找相似 atom
3. **Lazy Import 模式**：fetchers 用 __getattr__ 延迟导入，parser 不依赖浏览器
4. **性能基准**：Scrapling Parser 比 BS4 快 700-1600x，接近理论极限
5. **Adaptive Storage**：SQLite 存储元素指纹，网站变更后能 relocate

### 技术栈
Parser: lxml + cssselect + orjson
Fetchers: curl_cffi + playwright + patchright
Spiders: anyio
