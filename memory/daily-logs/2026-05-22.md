### 阶段2：状态机
**文件**: `memory/scripts/memory-governor.js`

实现了白皮书 v1.2 定义的完整状态机：

**核心公式**：
```
E_activation = [I × (1+C)/2] × S × ln(e + Σw_i × D_i)
```

**THRESHOLDS**：
- `TAU_ACTIVE = 0.55` — E above this → ACTIVE
- `TAU_CRYO = 0.25` — E below this → CRYOSLEEP
- `K_CONFLICT = 0.75` — cognitive conflict → RESEARCH
- `D_CLUSTER = 5` — cluster size to trigger distillation
- `BOUNDARY_GATE = 0.38` — gating below this → CRYOSLEEP

**6 个 test case 全pass**：
- high E → ACTIVE ✓
- low E → CRYOSLEEP ✓
- K > 0.75 → RESEARCH ✓
- cluster ≥ 5 + CRYOSLEEP → DEPRECATED ✓
- validation failed → DEPRECATED ✓
- gating 0.3 → CRYOSLEEP ✓

**边界门控修复**：`allowedNamespaces` match 后应返回 1.0 而非 0.5
